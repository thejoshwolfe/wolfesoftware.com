Professional Work

DDC-I, Inc.

December 2008 - January 2015

I was the architect and majority contributor for OpenArbor®, an Eclipse-based IDE in Java for developing C/C++ applications for our safety-critical RTOS, Deos™.

  • Wrote a Makefile generator to implement all of the IDE's build steps in a way that is consistent inside and outside the GUI.
  • Wrote an application/library interdependency system that manages build-time integration, system-wide metadata configuration, and deploy-time file system management.
  • Wrote an options specification system to allow general build options and options specific to certain target architectures and debug/production variants.
  • Wrote a Gantt chart viewer that shows the timeline of when each thread ran on the embedded system.
  • Extended the debugger interface to support lazy pointer following, variable byte sizes, and more.

I also wrote and maintained dozens of internal tools written in Python that automated tasks such as running tests and indexing documentation.
